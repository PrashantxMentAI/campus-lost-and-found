# **App Name**: Campus Connect: Lost & Found

## Core Features:

- Submission Form: A simple form for users to submit lost or found items with details like item type, name, location, and contact information.
- Firestore Integration: Store submitted items in Google Firestore with real-time updates to the item list.
- Item Listing: Display lost and found items in a list format with clear item details.
- Item Auto-Categorization: Automatically categorize items (e.g., Mobile, Wallet, ID Card, Keys, Bag, Other) using an AI tool based on the item description.
- Smart Search: Implement a semantic search function to match items by meaning, not just exact words.
- Contact Information Display: Clearly display contact information for each item to facilitate easy communication between finders and losers.

## Style Guidelines:

- Primary color: Light, desaturated blue (#B0E2FF) to evoke a sense of calm and reliability.
- Background color: Very light blue (#E6F7FF), almost white, to provide a clean and simple backdrop.
- Accent color: Soft green (#BCE29E) to highlight interactive elements and important information, creating visual interest.
- Font: 'PT Sans', a humanist sans-serif suitable for both headlines and body text, ensuring readability and a modern feel.
- A simple, single-column layout optimized for mobile and desktop, ensuring a beginner-friendly and clean presentation.
- Use clear, minimalistic icons to represent item categories, making it easy to identify items at a glance.
- Subtle animations, like fade-in effects on new items, to enhance the user experience without overwhelming the interface.
